# **App Name**: EduVault

## Core Features:

- Curated NCERT Notes: Provide concise, downloadable NCERT notes for classes 9-12, covering all subjects.
- Important Questions Bank: Offer a comprehensive collection of important questions for each chapter, categorized by class and subject.
- Solved PYQs: Include solved Previous Year Questions (PYQs) for CBSE classes 10 and 12, and for JEE/NEET exams.
- Competitive Exam Materials: Provide dedicated sections for JEE and NEET, with notes, PYQs, formula sheets, and mock tests.
- Smart Search: Enable users to quickly find specific chapters, topics, or materials using a dynamic search bar.
- Personalized Dashboard: Allow users to save notes, track progress, and access interactive quizzes for personalized learning.
- AI Study Guide tool: Using a local LLM to reason over all of the notes to suggest which material will give the user the best explanation for a concept they're having trouble understanding.

## Style Guidelines:

- Primary color: Neon green (#39FF14) to highlight interactive elements and CTAs, creating a vibrant and engaging experience.
- Background color: Light-black (#1c1c1c) to provide a modern and sleek look that is easy on the eyes.
- Accent color: Blue (#007BFF) used for secondary interactive elements and links, providing visual contrast and indicating interactive elements.
- Body and headline font: 'Inter', a sans-serif font known for its clean and modern appearance, is used for both headings and body text to maintain readability and consistency throughout the platform.
- Utilize a set of consistent, outline-style icons to represent different sections, topics, and actions, enhancing visual appeal and usability.
- Employ a card-based layout to organize content, making it easy to browse and access different study materials.
- Implement subtle animations, such as smooth transitions and hover effects, to provide feedback and enhance user interaction.
